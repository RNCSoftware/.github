# RNCSoftware

**Precision software tools for real-time systems engineers and laboratory automation specialists.**

Built by an engineer with decades of hands-on experience in real-time operating systems, instrument control, and high-volume laboratory analyzers — where timing, accuracy, and reliability are non-negotiable.

---

## Philosophy

- **Real-World Focus** — Designed around the actual workflows of RTOS and embedded engineers, not theoretical use cases. Every feature earns its place.
- **Desktop-First** — Native desktop applications. No browser required, no cloud dependency — your data stays on your machine.
- **Precision by Design** — Tools that reflect the precision expected in real-time systems engineering: deterministic behavior, no surprises.
- **Open Development** — Source available on GitHub. Built iteratively with transparency — you can see exactly what's in the software you're running.

## Products

### PTTimeline — Process-Task Timeline Editor & Visualizer

A desktop suite for creating, editing, and visualizing process-task timing diagrams. Targets embedded and RTOS engineers who need to document, analyze, and communicate process scheduling and task timing with precision.

| Application | Description |
|-------------|-------------|
| **PTTEdit** | Timeline data editor — define processes, tasks, durations, and formula-driven scheduling |
| **PTTPlot** | Chart renderer — Gantt-style timeline visualization with dependency arrows |
| **PTTView** | Output viewer — review and present exported diagrams |

📦 Windows &nbsp;·&nbsp; 🐍 Python / PySide6 &nbsp;·&nbsp; 📄 MIT License
